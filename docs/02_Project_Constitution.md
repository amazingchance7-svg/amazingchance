# Amazing Chance — Project Constitution

**Document Version:** 1.0  
**Status:** Draft  
**Last Updated:** 2026-07-21

---

# Purpose

Amazing Chance exists to build the world's most transparent jackpot platform.

The platform is designed so that users never have to trust the company.
Instead, every important action can be independently verified.

---

# Mission

Give every person an equal, transparent and verifiable opportunity to win.

---

# Vision

Become the global standard for transparent online jackpot platforms.

When people think:

"I want a jackpot platform I can verify myself."

they should think of Amazing Chance.

---

# Core Values

## 1. Transparency First

Everything that can be public should be public.

Examples:

- ticket count
- draw history
- winning numbers
- Random.org results
- verification certificates

---

## 2. Verifiability Over Trust

Users should never be forced to trust Amazing Chance.

Every draw must be independently verifiable.

---

## 3. Simplicity

Every feature must be understandable within seconds.

If a feature needs long explanations, it should be redesigned.

---

## 4. Fairness

Every valid ticket has exactly the same probability of winning.

No VIP users.

No hidden advantages.

No paid priority.

---

## 5. Security

Security is never sacrificed for speed.

Every financial operation must be auditable.

---

## 6. User Respect

No dark patterns.

No misleading timers.

No fake urgency.

No hidden fees.

---

# What Never Changes

These principles are permanent.

They cannot be removed without changing the identity of Amazing Chance.

- Transparent draws
- Public verification
- Equal chances
- Fixed prize distribution
- Public draw archive
- Independent randomness
- Honest communication

---

# What Can Change

These may evolve.

- UI
- Branding
- Features
- Technologies
- Payment providers
- Infrastructure
- Marketing

---

# Product Decision Rule

Every new feature must answer YES to all questions.

1. Does it increase trust?
2. Does it improve user experience?
3. Does it keep the system simple?
4. Can it be verified?
5. Can it scale?

If the answer is NO to any of them, the feature must be reconsidered.

---

# Long-Term Goal

Amazing Chance should become the benchmark for transparency in online jackpot platforms.

Success is measured not only by revenue, but also by the level of trust users place in the platform.
